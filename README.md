## 简介

### java常用的工具包

* 各种 util 类
* fqueue基于文件的队列，核心类拷贝自 [googlecode](https://code.google.com/archive/p/fqueue/)
* 图片验证码
* spring 日期转换
* spring bean validation异常处理
